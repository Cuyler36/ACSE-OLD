# ACSE - Animal Crossing Save Editor

ACSE is a save editor for the GameCube version of Animal Crossing.

## Features
* Player Editing
	* Name
	* Bells
	* Debt
	* Pockets
	* House (WIP)
* Acre Editing
* Town Editing
	* Items
	* Buildings
* Villager Editing
	* Villagers
	* Personalities
	* Catchphrases
	* Nicknames (Not Implemented)
